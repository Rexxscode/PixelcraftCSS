Fitur Utama

PixelCraft CSS dilengkapi dengan segala yang Anda butuhkan untuk membangun website bertema Minecraft

Authentic Pixel Design

Desain pixel-art yang autentik dengan border tebal, shadow block, dan efek 8-bit yang membuat website terlihat seperti dalam game Minecraft.

Pixel Art 8-bit Style

Complete Color System

16+ warna tema Minecraft dengan variasi gelap dan terang. Setiap warna memiliki kelas untuk background, text, dan border.

16 Colors Variations

Ready-to-Use Components

Button, card, navbar, modal, carousel, dan 20+ komponen lain yang sudah didesain dengan tema Minecraft. Responsif dan mudah dikustomisasi.

24+ Components Responsive

Fully Responsive

Dibangun dengan sistem grid responsif yang bekerja sempurna di semua perangkat. Mobile-first approach dengan breakpoints yang jelas.

Mobile First Flexbox Grid

Utility-First Approach

100+ utility class untuk spacing, warna, typography, flexbox, grid, dan efek khusus. Bangun desain cepat tanpa menulis CSS custom.

Utility-First Rapid Development

Game-Inspired Elements

Health bar, heart containers, texture patterns, dan elemen khusus bertema game untuk pengalaman yang imersif dan interaktif.

Game Elements Immersive
